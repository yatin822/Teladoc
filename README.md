# Pre-Conditions

1. Install JDK 8
2. Install Intellji
3. Clone project to your computer

# Steps to Run Project

1. Build Maven 
2. Go to directory src/test/java/testRunner/
3. Right Click on Runner file and select the option Run 'Runner'
